# 项目简报：LLM Sim

## 项目目标

在 SUMO GUI 中运行广州珠江新城/天河 CBD 小范围真实地图交通仿真，生成 1000 名司机的一日出行，并展示早晚高峰、事故限速、大型活动散场、路口 Agent 监测和车辆重规划变色。

## 用户需求

- 保留并规范现有仿真项目。
- 项目根目录命名为 `project000_LLM_Sim`。
- 采用全局工作台规定的标准目录结构。

## 交付物

- 可运行的 SUMO GUI 交通仿真代码与配置。
- 示例地图、路线、活动链和出行数据。
